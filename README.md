## 🚀 Habit Tracker ML Project

## 📌 Project Overview

This project is a Machine Learning-based Habit Tracker that predicts whether a user will complete their daily habit based on lifestyle factors like sleep, mood, and stress.

The goal is to help users improve consistency and build better habits using data-driven insights.

---

## 🎯 Problem Statement

Many people struggle to maintain daily habits. This project aims to:

* Predict habit completion
* Identify key factors affecting consistency
* Provide actionable insights

---

## 📊 Dataset Description

The dataset includes user lifestyle and behavior data.

### Key Features:

* Sleep_Hours
* Stress_Level
* Mood_Score
* Daily_Routine_Score
* Screen_Time
* Exercise_Minutes

### Target Variable:

* Habit_Completed_Today (0 or 1)

---

## 🧠 Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)

---

## 📈 Model Performance

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 90%      |
| Decision Tree       | 88%      |
| Random Forest       | 91%      |
| KNN                 | 68%      |
| XGB                 | 94%      |
